Lightprint Engine

Веб-приложение для передачи и приёма данных с использованием видимого света (VLC). Используйте камеру и экран вашего устройства для аутентификации и обмена информацией.

Ключевой Функционал

Приёмник (Receiver)

Использует камеру устройства (`getUserMedia`) для захвата световых сигналов. Анализирует мерцание, стабильный свет и QR-коды с цветовой модуляцией для приёма данных.

Передатчик (Transmitter)

Генерирует световые сигналы для передачи текста или файлов. Использует экран для мигания (Manchester) или отображения серии цветных QR-кодов.

Поддерживаемые Режимы

Flicker (Мерцание)

Аутентификация по шаблону мерцания. Сравнение сигналов через DTW и кросс-корреляцию.

Stable (Стабильный)

Распознавание объектов (стабильный источник света) с использованием MobileNet (TF.js) для извлечения признаков.

Data (Manchester)

Передача данных с помощью Manchester-кодирования. Использует CRC8 для проверки целостности данных.

Data QR (Цветной QR)

Передача файлов/текста через QR с цветовой модуляцией (8 цветов, 3 бита на зону) и AES-шифрованием.
